# 🥁 Gitpal — Ace the palace AI

> The ambient AI companion that travels with you across GitHub, surfaces pinned content, and guides your next professional move.

Gitpal is the first module of the **Git AI Suite** — four specialized AI agents engineered to master every dimension of your GitHub workflow.

---

## ✨ Features

- **🥁 Floating orb assistant** — a draggable conversation bubble that follows you and responds contextually
- **📌 Pin suggestions** — AI-curated list of repos and projects worth pinning to your profile
- **🚀 Pro Moves coaching** — actionable career-level advice surfaced in real time
- **🌐 Dashboard** — overview of all four AI agents and their status

## 🚀 Getting Started

No build step required — it's a static web app.

```bash
# Clone the repo
git clone https://github.com/www-infinity4/Gitpal-.git
cd Gitpal-

# Open in your browser
open index.html
# or serve locally:
npx serve .
```
